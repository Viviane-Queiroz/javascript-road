Starting with DOM

What is it?

Basically, each browser has a Document Object Model - DOM - built in it since this technology supports the objects that describe a general document and it's own objects.

In this way, the DOM give the fundamental tools so that script languages and even HTML could be used to create a web page as we know it.

It's very common to point a "DOM tree" to better understand the parenthood relation we observe in DOM.
