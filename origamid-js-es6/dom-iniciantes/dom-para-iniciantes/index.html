Document Object Model (DOM)

Dom is an interface that represents HTML and XLML documents through objects.

With this interface native in browsers it is possible to manipulate the structure, style and content of those documents.


Some of DOM characteristics:


  • "window" is the global object present in DOM, so it's use is not mandatory
  • An object inside DOM can have property and methods
  • Window and Element are the most common objects used
  • DOM helps to give a web page dynamism since it allow script languages as JS work with the HTML content
